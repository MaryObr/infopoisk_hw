# infopoisk_hw
homework for infosearch course

В папке hw1 файлы для домашнего задания 1, поиск осуществляется из файла usage.ipynb в функции general_search.

Я буду использовать параллельный корпус упрощенных вариантов русских текстов RuAdapt, а именно упрощенные варианты текстов, так как он имеет разбивку по предложениям, содержит разнообразную лексику (на распределение может влиять специфика упрощения) и небазовую стилистику (тексты энциклопедичиские).
